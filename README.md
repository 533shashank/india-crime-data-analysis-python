# 🇮🇳 India Crime Data Analysis (2001–2012)

This project analyzes district-wise IPC crime data in India using Python.  
The dataset is sourced from the National Crime Records Bureau (NCRB).

---

## 📊 Project Overview

The goal of this project is to explore crime patterns across India and extract meaningful insights using data analysis and visualization techniques.

---

## 🔍 Features

- Data cleaning and preprocessing of real-world dataset  
- Exploratory Data Analysis (EDA)  
- State-wise and district-wise crime analysis  
- Crime trend analysis over years (2001–2012)  
- Identification of top crime-prone states and districts  
- Visualization using graphs and heatmaps  

---

## 📁 Dataset

- Source: NCRB (National Crime Records Bureau), Government of India  
- Data Type: District-wise IPC crimes  
- Years Covered: 2001 – 2012  

---

## 🛠 Technologies Used

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

---

## 📈 Key Insights

- Certain states consistently report higher crime rates  
- Crime trends show noticeable growth over the years  
- Specific crime categories have increased significantly  
- District-level analysis helps identify crime hotspots  

---

## 📷 Visualizations

The project includes:
- Line charts (trend analysis)  
- Bar charts (state comparisons)  
- Heatmaps (state vs year patterns)  

---

## 💡 Learning Outcomes

- Handling real-world messy datasets  
- Data cleaning and preprocessing techniques  
- Data visualization best practices  
- Extracting meaningful insights from raw data
